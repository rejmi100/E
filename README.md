# 📋 Třídní nástěnka – Úkoly

Jednoduchá webová aplikace pro evidenci školních úkolů vytvořená v rámci školního projektu.

## 🚀 Funkce
- Přidávání úkolů (Předmět, Popis, Termín).
- Ochrana proti prázdnému zadání.
- **Bonus:** Ukládání úkolů do prohlížeče (localStorage).

## 🛠 Jazyky
- HTML5
- CSS
- JavaScript
